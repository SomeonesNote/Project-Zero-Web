# 작업 완료 체크리스트

## 코드 품질 검사
1. `npm run lint`: ESLint 검사 통과
2. `npm run format`: Prettier 포맷팅 적용
3. `npm test`: 테스트 통과
4. `npm run build`: 빌드 성공

## 기능 검증
1. 개발 서버 실행: `npm run dev`
2. 브라우저에서 기능 동작 확인
3. 반응형 디자인 검증
4. 콘솔 에러 없음 확인

## 코드 리뷰 포인트
1. TypeScript 타입 안전성
2. 컴포넌트 재사용성
3. 이벤트 처리 적절성
4. 메모리 누수 방지 (이벤트 리스너 정리)
5. 에러 핸들링

## Git 관련
- 커밋 전 lint/test 통과 확인
- 의미있는 커밋 메시지 작성
